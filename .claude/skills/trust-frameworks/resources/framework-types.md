# Framework Types

## 1. ACRONYM FRAMEWORK
**Pattern:** Each letter = step or principle
**Examples:** S.M.A.R.T. goals, AIDA, SWOT
**Best for:** Memorable, shareable, fits in social posts

## 2. NUMBERED SYSTEM
**Pattern:** X steps, pillars, or principles
**Examples:** 7 Habits, 4-Hour Workweek, 12 Rules
**Best for:** Comprehensive guides, courses

## 3. MATRIX/QUADRANT
**Pattern:** 2x2 grid categorization
**Examples:** Eisenhower Matrix, BCG Matrix
**Best for:** Decision-making, categorization

## 4. LADDER/PYRAMID
**Pattern:** Hierarchical levels
**Examples:** Maslow's Hierarchy, Content Pyramid
**Best for:** Progression, skill levels

## 5. CYCLE/WHEEL
**Pattern:** Iterative process
**Examples:** PDCA, Customer Lifecycle
**Best for:** Ongoing processes, feedback loops

## 6. EQUATION/FORMULA
**Pattern:** A + B = C
**Examples:** E=mcÂ², Value = Benefits - Cost
**Best for:** Simple concepts, viral sharing
