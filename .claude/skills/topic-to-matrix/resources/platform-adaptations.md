# Platform-Specific Adaptations

## 1. EMAIL (800-1200 words)
- **Hook:** First 2 sentences must grab attention
- **Body:** Story → Lesson → Application
- **CTA:** One clear next action
- **Subject Line:** 5-7 word curiosity driver

## 2. BLOG POST (1200-1800 words)
- SEO-optimized title
- Introduction with hook
- H2/H3 subheadings for structure
- Bullet points for scannability
- Images/graphics
- Conclusion with CTA

## 3. TWITTER/X THREAD (8-15 tweets)
- **Tweet 1:** Pattern interrupt hook
- **Tweets 2-3:** Setup/context
- **Tweets 4-12:** Numbered points or story beats
- **Tweet 13-14:** Summary/takeaway
- **Final Tweet:** CTA + "Retweet if valuable"

## 4. TWITTER/X POST (Single, 280 chars)
- One punchy insight
- Emotionally charged
- Pattern interrupt or curiosity
- Stand-alone value

## 5. LINKEDIN POST (150-500 words)
- Professional tone but authentic
- Story or insight opening
- 3-5 key points with line breaks
- CTA at end (comment, DM, link)
- 3-5 relevant hashtags

## 6. INSTAGRAM CAPTION (300-500 words)
- Hook in first line (visible before "more")
- Story or relatable content
- Bullet points with emojis
- Strong CTA
- 10-15 hashtags

## 7. INSTAGRAM CAROUSEL (5-8 slides)
- **Slide 1:** Hook title + visual
- **Slides 2-6:** One key point per slide
- **Slide 7:** Summary
- **Slide 8:** CTA + account tag
- Design: Bold text, simple backgrounds

## 8. YOUTUBE VIDEO (8-15 minutes)
- **0:00-0:30** - Pattern interrupt hook
- **0:30-1:00** - Context/setup
- **1:00-10:00** - Main content (3-5 key points)
- **10:00-12:00** - Recap + transformation promise
- **12:00-13:00** - CTA (subscribe, link, offer)
- Title: 60-70 chars, keyword-optimized
- Description: First 150 chars visible, include CTA

## 9. YOUTUBE SHORT / TIKTOK (60 seconds)
- **0-3 sec:** Hook (attention grab)
- **3-15 sec:** Setup (1-2 sentences)
- **15-50 sec:** Main content (3-5 quick points)
- **50-60 sec:** CTA (follow/like/comment)
- Captions: On-screen text for key points
- Hashtags: 5-10 relevant

## Voice Adaptation by Platform
- **Email:** Most intimate, longest form, deepest vulnerability
- **Blog:** Educational, structured, SEO-aware but authentic
- **Twitter:** Punchy, opinionated, conversation-starter
- **LinkedIn:** Professional but human, authority-building
- **Instagram:** Visual-first, story-driven, community-focused
- **YouTube:** Teaching mode, charismatic, longer attention span
