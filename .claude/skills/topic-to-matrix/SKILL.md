# Topic to Matrix Skill

## Purpose
Transform one topic into a multi-platform content matrix with 19 distinct pieces across different formats, platforms, and ERISE buckets.

## Your Role
Content expansion engine using the ERISE content multiplication system.

## Process

### Step 1: Get Input
Ask for:
1. Topic to expand (from Topic Library or new idea)
2. Brand Profile path
3. Voice Meta prompt (if available)

### Step 2: Topic Analysis
Determine:
- **Primary ERISE Bucket:** Educate / Relate / Inspire / Sell / Engage
- **Core Message:** ONE key insight or value proposition
- **Awareness Level:** Best suited level
- **Emotional Hook:** Curiosity / Fear / Hope / Anger / Relief / etc.
- **Transformation Promise:** What audience gains

### Step 3: ERISE Mapping
**Read:** `resources/erise-mapping.md`

Select best variables (5 from 25) based on primary bucket.
Match to suggested formats.

### Step 4: Multi-Platform Matrix
**Read:** `resources/platform-adaptations.md`

Generate variations across 8 platforms:
1. EMAIL (800-1200 words)
2. BLOG POST (1200-1800 words)
3. TWITTER/X THREAD (8-15 tweets)
4. TWITTER/X POST (280 chars)
5. LINKEDIN POST (150-500 words)
6. INSTAGRAM CAPTION (300-500 words)
7. INSTAGRAM CAROUSEL (5-8 slides)
8. YOUTUBE VIDEO (8-15 minutes)

### Step 5: Content Multiplication
From ONE topic, generate:

**Primary Asset (1):** Blog / Email / YouTube / Podcast

**Atomized Assets (18):**
- 3 Twitter threads (different angles)
- 5 Single tweets (best quotes)
- 2 LinkedIn posts (professional + story)
- 3 Instagram posts (visual varieties)
- 1 Instagram carousel (5-8 slides)
- 1 YouTube Short (60 sec)
- 1 TikTok (30-60 sec)
- 2 Email follow-ups

**Total: 19 pieces from ONE topic**

### Step 6: Voice Integration
If Voice Meta provided:
1. Reference Voice DNA patterns
2. Check Disgust Mapper boundaries
3. Apply signature phrases
4. Match energy for each format
5. Honor contrarian angles

### Step 7: Output Format
**Read:** `resources/output-template.md`

## Important Rules
1. Platform-specific adaptation
2. Voice consistency
3. Strategic variety
4. Actionable specificity
5. CTA diversity
6. Honor boundaries
7. Respect constraints
8. Format-appropriate length

Ask for topic and Brand Profile/Voice Meta.
