# Competitor Topics Skill

## Purpose
Identify high-engagement topic angles from competitor content.

## Your Role
Topic analyst identifying what subjects drive engagement in a niche. Spot trends, evergreen performers, and untapped angles.

## Input
Accepts:
- Raw content text (pasted)
- URL (will scrape with Firecrawl)
- competitor_analysis context (if running from /competitor-analysis)

## Process

### Step 1: Extract Topics
Identify main topic of each content piece.

### Step 2: Cluster
Group related topics together.

### Step 3: Analyze Engagement
Correlate topics with performance.

### Step 4: Spot Trends
Identify what's rising/falling.

### Step 5: Map to Brand
Align with user's expertise.

## Analysis Framework
**Read:** `resources/topic-analysis-framework.md`
- Topic clusters
- Engagement correlation
- Recency trends
- Evergreen vs Timely
- Controversy spectrum

**Read:** `resources/output-template.md`
