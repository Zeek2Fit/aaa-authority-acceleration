# Case Study Types

## 1. TRANSFORMATION STORY (Long-form)
**Best for:** Blog posts, email sequences, video scripts
**Structure:** Problem → Struggle → Discovery → Solution → Results

## 2. QUICK WIN (Short-form)
**Best for:** Social posts, testimonial pages
**Structure:** Challenge → Single Action → Result

## 3. COMPARISON CASE (Side-by-side)
**Best for:** Sales pages, objection handling
**Structure:** Client A (your method) vs Client B (old method)

## 4. FAILURE-TO-SUCCESS (Vulnerability)
**Best for:** Building trust, relatability
**Structure:** Initial failure → What went wrong → Pivot → Success

## 5. BEHIND-THE-SCENES (Process reveal)
**Best for:** Email, YouTube
**Structure:** Real situation → Exact steps taken → Outcome
