# Trust Signal: Case Studies Skill
**E-E-A-T Category: EXPERIENCE**

## Purpose
Generate case study templates that demonstrate real-world results and transformations.

Case studies are your strongest EXPERIENCE signal. They prove you've done the work, gotten results, and can do it again.

## Input
Requires (auto-loaded):
- Brand Profile (transformation journey, methodology)
- Voice Meta (writing style)
- Competitor Analysis (winning structures) - optional

## Case Study Types
**Read:** `resources/case-study-types.md`
1. Transformation Story (Long-form)
2. Quick Win (Short-form)
3. Comparison Case (Side-by-side)
4. Failure-to-Success (Vulnerability)
5. Behind-the-Scenes (Process reveal)

## Output
**Read:** `resources/template-structure.md`

Generate 3-5 templates per type with:
- Hook options (based on voice + competitor hooks)
- Fill-in-the-blank template
- Call to action options
- Metrics to collect

## Voice Adaptation
Apply Voice Meta patterns:
- Use signature phrases in transitions
- Match sentence length patterns
- Include forbidden word avoidance
- Maintain emotional tone from profile

`trust_signals/case_studies/`
